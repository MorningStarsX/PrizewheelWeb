## 如何使用

### [1]使用在线网站
   [PrizewheelWeb](https://morningstarsx.github.io/PrizewheelWeb/)

#### 1.更改名单
- 点击`选择文件`按钮
  导入`json`文件
    文件格式:
    ```
    {
    "A":1,
    "B":2,
    ...(依此类推)
    }
    ```
 >使用前给个Star吧

### [2]下载HTML文件本地运行
- 文件路径
   ```
   web/index.html
   ```
   进入该文件点击下载按钮
- 按照`更改名单`步骤进行
